# Forge2 Qualifier - Uttam Maurya

## Project Overview

A Trello-style Kanban Board built using Laravel (Backend) and React (Frontend).

## Tech Stack

* Laravel 13
* React + Vite
* SQLite
* OpenClaw
* Slack
* Gemini 2.5 Flash

## Features

* Boards
* Lists (Todo, Doing, Done)
* Cards
* Due Dates
* Member Assignment
* Tags

## Run Backend

```bash
cd backend
composer install
php artisan migrate
php artisan serve
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
